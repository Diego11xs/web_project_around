# Tripleten web_project_around

🌎 Around The U.S.
🧾 Descripción

Around The U.S. es una página web responsiva creada como parte del Sprint 7 del programa de desarrollo web.
El proyecto presenta un perfil de usuario editable mediante un popup interactivo hecho con JavaScript, además de una galería de lugares emblemáticos con un diseño limpio y moderno.

🚀 Funcionalidades

🖋️ Editar perfil: Permite modificar el nombre y la ocupación del usuario a través de un formulario emergente.

💾 Guardar cambios: Los datos ingresados se reflejan dinámicamente en la interfaz sin recargar la página.

❌ Cerrar popup: Se puede cerrar el popup con el botón de cierre o al guardar los cambios.

📱 Diseño responsivo: El sitio se adapta correctamente a diferentes tamaños de pantalla.

🧩 Tecnologías utilizadas

HTML5 → estructura semántica del contenido.

CSS3 → uso de Flexbox, Grid y responsive design.

Normalize.css → para asegurar consistencia visual entre navegadores.

JavaScript (DOM) → manipulación dinámica del contenido y manejo de eventos.

💡 Aprendizajes clave

Durante el desarrollo de este sprint, se reforzaron los siguientes conceptos:

Uso del DOM para seleccionar y modificar elementos HTML.

Implementación de event listeners (click, submit).

Prevención del comportamiento por defecto con event.preventDefault().

Manejo de formularios dinámicos y actualización del contenido sin recargar.

Aplicación de modificadores de clase (.popup_opened) para mostrar u ocultar componentes.
